Contoh Penggunaan Phyton

Berikut adalah contoh penggunaan Python:

  • Program untuk menampilkan "Hello, World!" pada konsol: print("Hello, World!")
  • Program untuk menjumlahkan dua angka yang dimasukkan oleh pengguna: angka1 = float(input("Masukkan angka pertama: ")) angka2 = float(input("Masukkan angka kedua: ")) sum = angka1 + angka2 print("Hasil penjumlahan {0} dan {1} adalah {2}".format(angka1, angka2, sum))
  • Program untuk menghitung akar kuadrat dari angka yang dimasukkan oleh pengguna: angka = float(input("Masukkan angka: ")) akar = angka ** 0.5 print("Akar kuadrat dari {0} adalah {1}".format(angka, akar))
  • Program untuk menukar nilai dari dua variabel: x = input("Masukkan nilai x: ") y = input("Masukkan nilai y: ") tukar = x x = y y = tukar print("Setelah ditukar, nilai x adalah {0} dan nilai y adalah {1}".format(x, y)) Program untuk menentukan apakah sebuah bilangan adalah bilangan prima atau tidak: angka = int(input("Masukkan angka: ")) if angka > 1: for i in range(2, angka): if (angka % i) == 0: print(angka, "bukan bilangan prima") break else: print(angka, "adalah bilangan prima") else: print(angka, "bukan bilangan prima")

Python

Python adalah bahasa pemrograman yang dikembangkan oleh Guido van Rossum pada tahun 1991 di CWI, Amsterdam sebagai kelanjutan dari bahasa pemrograman ABC. Python adalah bahasa pemrograman tingkat tinggi, yang ditafsirkan dan mudah dipelajari. Python memiliki sintaks yang mudah dibaca dan dipahami, sehingga memudahkan pengembang untuk menulis kode program dengan ringkas. Python juga memiliki library yang sangat banyak dan luas, yang memuat beragam perlengkapan dan fungsionalitas, sehingga kemudahan membangun program menjadi salah satu yang ditawarkan oleh Python. Python banyak digunakan dalam aplikasi web, pengembangan perangkat lunak, ilmu data, dan machine learning (ML). Python juga digunakan untuk mengotomatiskan tugas dan melakukan analisis data. Python merupakan bahasa pemrograman yang sangat populer dan banyak digunakan oleh para pengembang di seluruh dunia

Fungsi utama Python adalah sebagai bahasa pemrograman yang dapat digunakan untuk membangun situs web, mengotomatiskan tugas, melakukan analisis data, pengembangan perangkat lunak, dan machine learning (ML). Python juga digunakan untuk membuat aplikasi, otomatisasi, dan administrasi. Python memiliki banyak fungsi yang dapat digunakan untuk mengefisienkan program serta memudahkan programmer dalam menjalankan program besar. Fungsi dalam Python merupakan kumpulan perintah yang dikelompokkan menjadi satu, sehingga memungkinkan programmer untuk mengefisienkan penulisan program, terutama program besar. Dengan menggunakan fungsi, programmer dapat memecah program menjadi beberapa bagian serta fungsinya dipanggil sewaktu-waktu ketika dibutuhkan, sehingga membuat program lebih terstruktur. Beberapa fungsi Python yang sering digunakan antara lain: type(), enumerate(), dan eval()

Tag-Tag Phyton

  • color: Digunakan untuk mengatur warna teks.
  • print(): Digunakan untuk mencetak output ke konsol.
  • input(): Digunakan untuk menerima input dari pengguna.
  • if/else: Digunakan untuk membuat kondisi pada program.
  • text-align: Digunakan untuk mengatur posisi teks, seperti rata kiri, rata kanan, atau rata tengah.
  • for/while: Digunakan untuk membuat perulangan pada program.
  • def: Digunakan untuk mendefinisikan fungsi pada program.
  • import: Digunakan untuk mengimpor modul atau pustaka pada program.
  • try/except: Digunakan untuk menangani kesalahan atau error pada program.
  • class: Digunakan untuk membuat kelas pada program.float
  • with: Digunakan untuk membuka dan menutup file secara otomatis.
