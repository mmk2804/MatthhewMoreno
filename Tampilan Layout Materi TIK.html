Tag-tag HTML

  • !DOCTYPE: Tag untuk menentukan tipe dokumen."
  • html: Tag untuk membuat sebuah dokumen HTML.
  • title: Tag untuk membuat judul dari sebuah halaman.
  • body: Tag untuk membuat tubuh dari sebuah halaman.
  • h1 to h6: Tag untuk membuat heading.
  • p: Tag untuk membuat paragraf.
  • hr: Tag untuk membuat perubahan dasar kata didalam isi.
  • !--...--: Tag untuk membuat komentar.
  • b: Tag untuk membuat huruf bercetak tebal.
  • bdi: Mengisolasi bagian dari teks yang dapat diformat dalam arah yang berbeda dari teks lain di luarnya (tag baru HTML5).
  • center: Tag untuk membuat jajaran teks menjadi ditengah (tidak disupport lagi di HTML5).

CSS & HTML

HTML (Hypertext Markup Language) adalah bahasa markup standar yang digunakan untuk membuat struktur halaman website agar dapat ditampilkan pada web browser. HTML memungkinkan user membuat dan menyusun bagian heading, paragraf, blockquote, dan link atau tautan untuk halaman web serta aplikasi. HTML bukanlah bahasa pemrograman karena tidak bisa memberikan fungsi yang dinamis. HTML adalah standar Internet yang dikendalikan oleh World Wide Web Consortium (W3C). HTML pertama kali dibuat oleh Tim Berners-Lee, seorang ahli fisika di lembaga penelitian CERN yang berlokasi di Swiss pada tahun 1990. Versi pertama HTML terdiri dari 18 tag. Saat ini terdapat 140 HTML tag walaupun sebagian sudah tak didukung oleh versi terbaru browsernya. Fungsi utama HTML adalah untuk membangun struktur halaman web, mulai dari membuat header, footer, navigasi, hingga kontennya. HTML merupakan pondasi awal dalam menyusun kerangka halaman web secara terstruktur sebelum membahas terkait tampilan desain dan sisi fungsionalitas.

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan atau styling dari halaman web yang dibuat dengan HTML. CSS memungkinkan user untuk mengatur warna, font, ukuran, layout, dan efek visual lainnya pada elemen HTML. CSS memisahkan antara konten dan tampilan pada halaman web, sehingga memudahkan user untuk mengubah tampilan halaman web tanpa harus mengubah kontennya. CSS pertama kali diperkenalkan pada tahun 1996 oleh World Wide Web Consortium (W3C). Fungsi utama CSS adalah untuk memperindah tampilan halaman web dan meningkatkan pengalaman pengguna. CSS juga memungkinkan user untuk membuat halaman web yang responsif dan dapat diakses pada berbagai perangkat, seperti desktop, tablet, dan smartphone. beberapa fungsi dan kegunaan CSS seperti Memperindah tampilan halaman web,Meningkatkan pengalaman pengguna, dan lain sebagainya.

Tag-Tag CSS

  • color: Digunakan untuk mengatur warna teks.
  • font-size: Digunakan untuk mengatur ukuran font.
  • font-family: Digunakan untuk mengatur jenis font.
  • background-color: Digunakan untuk mengatur warna latar belakang.
  • text-align: Digunakan untuk mengatur posisi teks, seperti rata kiri, rata kanan, atau rata tengah.
  • margin: Digunakan untuk mengatur jarak antara elemen dengan elemen lainnya.
  • padding: Digunakan untuk mengatur jarak antara konten dengan batas elemen.
  • border: Digunakan untuk mengatur garis batas pada elemen.
  • display: Digunakan untuk mengatur tampilan elemen, seperti inline, block, atau none.
  • float: Digunakan untuk mengatur posisi elemen, seperti mengapungkannya ke kiri atau ke kanan.
